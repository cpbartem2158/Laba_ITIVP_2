# EduPlatform

Образовательная веб-платформа для онлайн-обучения

## Описание проекта

EduPlatform — современная образовательная платформа, предоставляющая доступ к качественным онлайн-курсам по различным направлениям.

## Используемые технологии

- HTML5
- CSS3
- JavaScript (ES6+)
- Node.js
- npm

## Установленные зависимости

### Production
- lodash - библиотека утилитарных функций

### Development
- nodemon - автоматическая перезагрузка приложения
- prettier - форматирование кода
